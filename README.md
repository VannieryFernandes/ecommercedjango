# Ecommerce django - curso na udemy

## Pré-requisitos

django 2.0 ou ++

Virtual Env na máquina

## Instalação

Na pasta do virtual env ative o ambiente virtual

```bash
source bin/activate
```
Na pasta do projeto

```bash
pip install -r requirements.txt

./manage.py runserver
```
